## Tecnical Stack:
* UIKit
* UI in code
* NSLayoutConstraints
* CollectionView
* Grand Central Dispatch
* URLSession
* SVGKit
* Kingfisher


## API
[Yandex Weather API](https://yandex.ru/pogoda/b2b/console/api-page)

## Description 
#### The application requests weather from Yandex for ten cities with a population of over a million.
